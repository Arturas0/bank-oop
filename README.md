# PHP bank application OOP style

Application witten with PHP in OOP style. <br />

**PHP:** 8.0 <br />
**Database:** MariaDB <br />
**Seeder:** yes (seeders/bank.php) <br />
**Loggin credentials (after seeder was ran):** 
`username: Jonas, password: 123`.

## Implemented:
1. Register page; <br />
2. Loggin page; <br />
3. Create account; <br />
4. Delete account; <br />
5. Add funds; <br />
6. Deduct funds; <br />
7. Validation for inputs; <br />
8. IBAN account automatic generation; <br />
9. Messages after action was performed; <br />
10. Personal code validator. <br />
